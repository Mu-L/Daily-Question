# '+' 与 '~' 选择器有什么不同

::: tip Issue
欢迎在 Gtihub Issue 中回答此问题: [Issue 317](https://github.com/shfshanyue/Daily-Question/issues/317)
:::

::: tip Author
回答者: [shfshanyue](https://github.com/shfshanyue)
:::

- `+` 选择器匹配紧邻的兄弟元素
- `~` 选择器匹配随后的所有兄弟元素
